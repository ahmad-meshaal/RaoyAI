# راوي | Rawi ✍️🤖

[Arabic Version Below](#arabic-version)

Rawi is a premium, AI-powered writing platform designed specifically for storytellers and novelists. It combines professional planning tools with modern AI assistance to help you build immersive worlds, deep characters, and engaging chapters.

![Rawi Landing](public/hero.png)

## Features
- **Project-Based Organization**: Manage your novels into structured projects with dedicated dashboards.
- **Chapter Workflow**: Write, edit, and organize chapters with auto-save and AI-assisted drafting.
- **Deep Character Creation**: Build comprehensive profiles for your characters with custom avatars generated by AI.
- **AI-Powered Assistance**: Generate plots, character traits, and chapter drafts using advanced LLMs.
- **Public Library**: Share your completed works with an active community. Includes search and engagement (likes/views).
- **Professional Export**: Export your manuscripts in PDF format ready for publishing.
- **Modern UI**: A warm, paper-themed interface optimized for Arabic typography (Amiri & Noto Sans Arabic).

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Framer Motion, Radix UI.
- **Backend**: Node.js, Express, Passport.js (Auth), Session-based security.
- **Database**: PostgreSQL with Drizzle ORM.
- **AI**: OpenAI SDK / Generative AI integrations for text and images.

## Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ahmad-meshaal/RaoyAI.git
   cd RaoyAI
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   Create a `.env` file from the example:
   ```env
   DATABASE_URL=postgresql://user:pass@localhost:5432/rawi
   SESSION_SECRET=your_secret_key
   OPENAI_API_KEY=your_openai_key
   ```

4. **Prepare the database**:
   ```bash
   npm run db:push
   ```

5. **Start development server**:
   ```bash
   npm run dev
   ```

## Production & Deployment

This project is optimized for deployment on **Railway**:

1. **Connect GitHub**: Fork the repo and connect it to a new Railway project.
2. **Add Postgres**: Provision a Postgres instance in Railway.
3. **Environment Variables**:
   - `DATABASE_URL`: Automatically provided by Railway.
   - `SESSION_SECRET`: Set a strong random string.
   - `OPENAI_API_KEY`: Required for AI features.
4. **CI/CD**: GitHub Actions are configured in `.github/workflows/ci.yml` for automated testing.

راوي هي منصة كتابة احترافية مدعومة بالذكاء الاصطناعي، مصممة خصيصاً للقاصين والروائيين العرب. تدمج المنصة بين أدوات التخطيط التقليدية وتقنيات الذكاء الاصطناعي الحديثة لتساعدك في بناء عوالم خيالية متكاملة، وشخصيات عميقة، وفصول مشوقة.

## المميزات
- **تنظيم المشاريع الأدبية**: أدر رواياتك كمشاريع منظمة وسهلة الوصول.
- **إدارة الفصول**: واجهة كتابة مريحة مستوحاة من ملمس الورق تتيح لك التركيز على صياغة أحداثك.
- **بناء الشخصيات**: أدوات متقدمة لتطوير سمات الشخصيات مع إمكانية توليد صور تخيلية لها بالذكاء الاصطناعي.
- **مساعد الذكاء الاصطناعي**: توليد أفكار الحبكة، سمات الشخصيات، ومسودات الفصول بضغطة زر.
- **المكتبة العامة**: تصفح أعمال الكتاب الآخرين، ابحث عن رواياتك المفضلة، وتفاعل مع المجتمع.
- **تصدير احترافي**: حول مسوداتك إلى ملفات PDF جاهزة للنشر الورقي أو الإلكتروني.
